Test problems¶

Different manufactured testprolbmes problems solving \(-\Delta u_a = f\) on the unit interval

testproblems.power_function(alpha=0.51, plot_sol=False)¶

Babushka example: \(u=x^{\alpha}\) with \(0.5 < \alpha < 1.5\)

Parameters

  • alpha (float) – power coefficient, default value 0.51. We assume 0.5 < alpha < 1.5

  • plot_sol (bool) – plot solution, default=False

Returns

  • u_a (df.expression) – analytic solution

  • f (df.expression) – corresponding rhs

testproblems.steep_mollifier(beta=2.5, plot_sol=False)¶

Mollifier with steep gradients: \(u= \exp(-1/((1-\beta^2)(x-0.5)^2)) / \exp(-1)\).

Parameters

  • beta (float) – steepness coefficient, default value 2.5

  • plot_sol (bool) – plot solution, default=False

Returns

  • u_a (df.expression) – analytic solution

  • f (df.expression) – corresponding rhs

testproblems.plot_analytic_sol(u_a, f, fname='analytic_solution')¶

Plot analytic solution

Parameters

  • u_a (function) – analytic solution

  • f (function) – corresponding rhs

  • fname (str) – plot file name, default=analytic_solution
